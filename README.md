# Data Analytics & Forecasting | Swarovski Internship (Jun 2024 - Jul 2024)

This repository documents my internship project at **Swarovski, Pune**, where I worked on building 
**predictive models** and **analytics dashboards** to optimize production, demand, and inventory planning.

## 🚀 Highlights
- 📊 **Predictive Modeling**: Achieved **78% R²** and reduced MAE by **44%** on pearl yield/loss prediction.
- ⚡ **Performance Boost**: Accelerated training speed **81x**, delivering results in <5 seconds.
- 🔎 **Exploratory Data Analysis**: Uncovered 7-year production trends using correlation, time-series, and variance analysis.
- 🧠 **ML Pipeline**: Built end-to-end system with feature engineering, K-fold cross-validation, and hyperparameter tuning.
- 📈 **Business Insights**: Identified *‘Colour’* as a key driver of waste; presented results with interactive dashboards.

## 🛠️ Tech Stack
- **Languages**: Python, SQL  
- **Libraries**: Pandas, Numpy, Scikit-learn, Matplotlib, Seaborn  
- **Visualization**: Tableau, Plotly Dash  
- **Tools**: GitHub, AWS
